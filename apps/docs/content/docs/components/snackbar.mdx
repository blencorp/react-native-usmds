## Installation

```package-install
npx usmds add snackbar
```

## Usage

```tsx
import { Snackbar } from '@components/Snackbar';
import { useState } from 'react';

function App() {
  const [isVisible, setIsVisible] = useState(true);

  return (
    <Snackbar
      isVisible={isVisible}
      variant="success"
      message="Message sent successfully"
      onAction={() => console.log('Action clicked')}
      onDismiss={() => setIsVisible(false)}
    />
  );
}
```

## Styling

The Snackbar component uses Tailwind CSS classes for styling through class-variance-authority (cva). Default styling includes:

- Container width: 329px
- One-line height: 48px
- Two-lines minimum height: 70px
- Dark background color
- Rounded corners (4px)
- Consistent padding
- Light text color
- Bold action buttons

## Variants

The snackbar has two main variants:
- Success: Displays with a checkmark icon
- Error: Displays with an alert circle icon

And two layout options:
- One-line: Compact layout for short messages
- Two-lines: Expanded layout for longer messages

## Accessibility

The Snackbar component:
- Uses semantic structure for notifications
- Maintains proper text contrast on dark background
- Provides adequate touch targets for buttons
- Supports keyboard navigation
- Includes clear action labels
- Times out automatically to prevent screen clutter
