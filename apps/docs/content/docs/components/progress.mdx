---
title: Progress
description: Progress indicators express an unspecified wait time or display the length of a process.
---

## Installation

```package-install
npx usmds add progress
```

## Usage

```tsx
import { Progress } from '@components/Progress';

<Progress value={40} />
```


### Custom Colors
```tsx

  value={40}

  indicatorClassName="bg-blue-500"
/>
```

## Animated Progress

The Progress component includes built-in smooth animations when the value changes:

```tsx
function AnimatedProgress() {
  const [progress, setProgress] = useState(40);

  const randomize = useCallback(() => {
    setProgress(Math.floor(Math.random() * 100));
  }, []);

  return (
    <View className="gap-4">
      <Progress value={progress} />
      <Button onPress={randomize}>
        <Text>Randomize</Text>
      </Button>
    </View>
  );
}
```

## Accessibility

The Progress component:
- Uses semantic HTML `<progress>` element on web
- Provides ARIA attributes for screen readers
- Maintains WCAG color contrast requirements
- Includes smooth animations that respect reduced motion preferences
