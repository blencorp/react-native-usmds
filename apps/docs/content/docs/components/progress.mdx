## Usage

```tsx

<Progress value={40} />
```

## Props

### Progress

Extends `ProgressPrimitive.Root` props.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| value | `number` | - | The progress value between 0 and 100. |
| className | `string` | - | Additional classes for the root element. |
| indicatorClassName | `string` | - | Additional classes for the indicator element. |
