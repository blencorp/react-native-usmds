---
title: BiometricSignIn
description: A component for enabling biometric authentication options like Face ID and Touch ID.
---

## Installation

```bash
npx usmds add biometricsignin
```

## Usage

```tsx
import { BiometricSignIn } from '@components/BiometricSignIn';
import { useState } from 'react';

function App() {
  const [enabled, setEnabled] = useState(false);

  return (
    <BiometricSignIn
      checked={enabled}
      onCheckedChange={setEnabled}
      biometricType="faceId"
    />
  );
}
```

## Styling

The BiometricSignIn component uses Tailwind CSS classes for styling. The appearance can be customized using the `className` prop:

```tsx
<BiometricSignIn
  checked={enabled}
  onCheckedChange={setEnabled}
  className="my-custom-class"
/>
```

Default styling includes:

- Rounded borders with base ink color
- Consistent padding and spacing
- Checkbox with primary color when checked
- Disabled state styling

## Accessibility

The BiometricSignIn component:
- Uses `accessibilityRole="checkbox"` for proper semantic structure
- Supports keyboard navigation
- Maintains WCAG color contrast requirements
- Provides appropriate `accessibilityState` for checked and disabled states
- Includes clear, descriptive text labels

## State Management

The component uses controlled state pattern:

```tsx
import { useState } from 'react';

function BiometricSettings() {
  const [enabled, setEnabled] = useState(false);

  return (
    <BiometricSignIn
      checked={enabled}
      onCheckedChange={setEnabled}
      biometricType="faceId"
    />
  );
}
```
