```package-install
npx usmds add badge
```

## Usage

```tsx
import { Badge } from '@components/Badge';

<Badge label="New" />
```

## Styling

The Badge component comes with predefined styles for each variant:

### Default Style
- Background: Primary lighter color
- Border: Primary light color
- Text: Primary foreground color

### Success Style
- Background: Success lighter color
- Border: Success light color
- Text: Success dark color

### Warning Style
- Background: Warning lighter color
- Border: Warning light color
- Text: Warning dark color

### Info Style
- Background: Info lighter color
- Border: Info light color
- Text: Info dark color

### Destructive Style
- Background: Destructive color
- Border: Destructive color
- Text: Destructive foreground color

## Custom Styling

The Badge component can be customized using className prop:

```tsx
<Badge 
  label="Custom" 
  className="bg-blue-100 border-blue-200"
/>
```

## Layout

Badges are designed to be inline elements with:
- Rounded corners (rounded-full)
- Consistent padding (px-2.5 py-1)
- Centered content
- Small text size (text-xs)
- Medium font weight

## Accessibility

The Badge component:
- Uses semantic HTML elements
- Maintains WCAG color contrast requirements for all variants
- Provides clear visual distinction between different states
- Uses appropriate text sizes for readability
