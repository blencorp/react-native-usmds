## Props

### AlertDialog

Extends `@rn-primitives/alert-dialog` Root props.

| Prop | Type | Description |
|------|------|-------------|
| open | `boolean` | The controlled open state of the alert dialog. |
| onOpenChange | `(open: boolean) => void` | Event handler called when the open state changes. |
| defaultOpen | `boolean` | The open state when initially rendered. |

### AlertDialogTrigger

Extends `@rn-primitives/alert-dialog` Trigger props.

| Prop | Type | Description |
|------|------|-------------|
| asChild | `boolean` | Change the default rendered element for the one passed as a child. |

### AlertDialogContent

Extends `@rn-primitives/alert-dialog` Content props.

| Prop | Type | Description |
|------|------|-------------|
| portalHost | `string` | The host where the alert dialog content will be rendered. |
| className | `string` | Additional classes for styling. |

### AlertDialogHeader

| Prop | Type | Description |
|------|------|-------------|
| children | `React.ReactNode` | The header content. |
| className | `string` | Additional classes for styling. |

### AlertDialogTitle

Extends `@rn-primitives/alert-dialog` Title props.

| Prop | Type | Description |
|------|------|-------------|
| children | `React.ReactNode` | The title content. |
| className | `string` | Additional classes for styling. |

### AlertDialogDescription

Extends `@rn-primitives/alert-dialog` Description props.

| Prop | Type | Description |
|------|------|-------------|
| children | `React.ReactNode` | The description content. |
| className | `string` | Additional classes for styling. |

### AlertDialogFooter

| Prop | Type | Description |
|------|------|-------------|
| children | `React.ReactNode` | The footer content. |
| className | `string` | Additional classes for styling. |

### AlertDialogAction

Extends `@rn-primitives/alert-dialog` Action props.

| Prop | Type | Description |
|------|------|-------------|
| children | `React.ReactNode` | The action button content. |
| className | `string` | Additional classes for styling. |

### AlertDialogCancel

Extends `@rn-primitives/alert-dialog` Cancel props.

| Prop | Type | Description |
|------|------|-------------|
| children | `React.ReactNode` | The cancel button content. |
| className | `string` | Additional classes for styling. |

## Examples

### Delete Confirmation

```tsx
import { useState } from 'react';

function DeleteConfirmation() {
  const [open, setOpen] = useState(false);

  const handleDelete = () => {
    // Perform delete action
    console.log('Deleted!');
    setOpen(false);
  };

  return (
    <AlertDialog open={open} onOpenChange={setOpen}>
      <AlertDialogTrigger asChild>
        <Button variant="destructive">
          <Text>Delete Account</Text>
        </Button>
      </AlertDialogTrigger>
      <AlertDialogContent>
        <AlertDialogHeader>
          <AlertDialogTitle>Are you absolutely sure?</AlertDialogTitle>
          <AlertDialogDescription>
            This action cannot be undone. This will permanently delete your account
            and remove your data from our servers.
          </AlertDialogDescription>
        </AlertDialogHeader>
        <AlertDialogFooter>
          <AlertDialogCancel>
            <Text>Cancel</Text>
          </AlertDialogCancel>
          <AlertDialogAction onPress={handleDelete}>
            <Text>Delete</Text>
          </AlertDialogAction>
        </AlertDialogFooter>
      </AlertDialogContent>
    </AlertDialog>
  );
}
```

### Controlled State

```tsx
import { useState } from 'react';
import { View } from 'react-native';

function ControlledDialog() {
  const [open, setOpen] = useState(false);
  const [loading, setLoading] = useState(false);

  const handleConfirm = async () => {
    setLoading(true);
    // Simulate async operation
    await new Promise((resolve) => setTimeout(resolve, 2000));
    setLoading(false);
    setOpen(false);
  };

  return (
    <View>
      <Button onPress={() => setOpen(true)}>
        <Text>Open Controlled Dialog</Text>
      </Button>

      <AlertDialog open={open} onOpenChange={setOpen}>
        <AlertDialogContent>
          <AlertDialogHeader>
            <AlertDialogTitle>Processing...</AlertDialogTitle>
            <AlertDialogDescription>
              Please wait while we process your request.
            </AlertDialogDescription>
          </AlertDialogHeader>
          <AlertDialogFooter>
            <AlertDialogCancel disabled={loading}>
              <Text>Cancel</Text>
            </AlertDialogCancel>
            <AlertDialogAction onPress={handleConfirm} disabled={loading}>
              <Text>{loading ? 'Processing...' : 'Confirm'}</Text>
            </AlertDialogAction>
          </AlertDialogFooter>
        </AlertDialogContent>
      </AlertDialog>
    </View>
  );
}
```
