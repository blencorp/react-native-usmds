## Usage

The Banner component should be placed at the top of every page on government websites. It provides a consistent way to identify official government sites and educate users about .gov/.mil domains and HTTPS security.

### Basic Usage

```tsx
import { Banner } from '@/components/ui/banner';

<Banner variant="standard-app" />
```

The standard-app banner includes an interactive "Here's how you know" button that expands to show information about .gov/.mil domains and HTTPS security.

### Language Support

The banner supports both English and Spanish:

```tsx
// English (default)
<Banner variant="standard-app" language="en" />

// Spanish
<Banner variant="standard-app" language="es" />
```

### Domain Types

Choose between .gov (default) and .mil domains:

```tsx
// .gov domain (default)
<Banner variant="standard-app" domain="gov" />

// .mil domain (Department of Defense)
<Banner variant="standard-app" domain="mil" />
```

## Props

### Banner

Extends `View` props.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| variant | `'standard-app' \| 'welcome'` | `'standard-app'` | The visual style variant. |
| domain | `'gov' \| 'mil'` | `'gov'` | The domain type for official government websites. |
| language | `'en' \| 'es'` | `'en'` | The language for banner content (English or Spanish). |
| title | `string` | - | Custom title text (used with 'welcome' variant). |
| trailingIcon | `ReactNode` | - | Optional trailing icon element. |
| className | `string` | - | Additional classes for styling. |

## Features

### Interactive Accordion

The standard-app banner includes an interactive "Here's how you know" button that expands to reveal:

- **Domain Information**: Explains what .gov or .mil domains represent and which organizations can use them
- **Security Information**: Shows lock icon and explains HTTPS security
- **User Guidance**: Advises when it's safe to share sensitive information

### Multilingual Support

- **English (en)**: Default language for most U.S. government sites
- **Spanish (es)**: Full Spanish translations following USWDS guidelines

### Domain-Specific Content

- **.gov**: For general U.S. government organizations
- **.mil**: For U.S. Department of Defense organizations

### Accessibility

- Proper ARIA labels for screen readers
- Keyboard navigable accordion
- Theme-aware colors (light/dark mode support)
- High contrast colors meeting WCAG AA standards

## Best Practices

### When to Use

- **Government Websites**: Use on every page of official U.S. government websites
- **Top Placement**: Always place at the very top of the page, above the main navigation
- **Consistent Usage**: Keep the banner consistent across all pages of your site
- **Language Selection**: Match the banner language to your site's primary language
- **Domain Matching**: Use `.mil` variant only for Department of Defense sites

### Accessibility Considerations

- The banner uses semantic HTML and proper ARIA labels
- Interactive elements are keyboard accessible
- Color contrast meets WCAG AA standards
- Screen reader friendly with descriptive labels

### Content Guidelines

According to USWDS guidelines:
- Do not modify the official text content
- Keep the banner visible (do not allow users to dismiss it)
- Avoid adding excessive visual styling that may distract from the message
- Ensure the flag icon and other visual elements remain visible

## API Reference

### Content Localization

The banner uses the `getBannerContent()` helper function internally to provide localized content. All text is automatically updated based on the `domain` and `language` props.

**Supported Combinations:**
- `gov` + `en` - U.S. Government (.gov) in English
- `gov` + `es` - U.S. Government (.gov) in Spanish
- `mil` + `en` - Department of Defense (.mil) in English
- `mil` + `es` - Department of Defense (.mil) in Spanish
