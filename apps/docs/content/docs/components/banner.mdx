## Installation

```package-install
npx usmds add banner
```

## Usage

```tsx
import { Banner } from '@components/Banner';

// Standard App Banner
<Banner
  variant="standard-app"
  title="An official app of the United States government"
  link="Here's how you know"
/>

// Welcome Banner
<Banner
  variant="welcome"
  title="Good afternoon, Daniel"
  trailingIcon={<Bell color="#1B1B1B" size={24} />}
/>
```

## Variants

### Standard App Banner

The standard app banner identifies official U.S. government applications.

```tsx
<Banner variant='standard-app' title='An official app of the United States government' link="Here's how you know" />
```

Specifications:

- Fixed height: 88px
- Maximum width: 393px
- Background color: base-lighter
- Text size: 12px (line height: 14px)
- U.S. flag icon (31x22px)
- Vertical divider (1px width, 56px height)
- "An official app" portion is always bold
- Link text is underlined and uses primary color

### Welcome Banner

The welcome banner provides a personalized greeting with optional icon.

```tsx
<Banner variant='welcome' title='Good afternoon, Daniel' trailingIcon={<Bell color='#1B1B1B' size={24} />} />
```

Specifications:

- Fixed height: 40px
- Fixed width: 329px
- Text size: 22px (line height: 28px)
- Bold font weight
- Title width: 273px
- Icon container: 40x40px with 8px padding
- No background color

## Accessibility

The Banner component includes:

- `accessibilityRole="header"` for semantic structure
- `accessibilityRole="link"` for clickable links
- `accessibilityLabel` for link elements
- Single-line text truncation with `numberOfLines={1}`
- Proper color contrast ratios

## Styling

The Banner uses Tailwind CSS through class-variance-authority (cva). Custom styles can be added via the `className` prop:

```tsx
<Banner variant='standard-app' title='An official app of the United States government' className='my-custom-classes' />
```
