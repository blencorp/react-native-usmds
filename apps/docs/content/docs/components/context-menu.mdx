## Usage

```tsx
import {
  ContextMenu,
  ContextMenuContent,
  ContextMenuItem,
  ContextMenuTrigger,
} from '@/components/ui/context-menu';
import { Text } from '@/components/ui/text';

<ContextMenu>
  <ContextMenuTrigger>
    <Text>Right click or long press</Text>
  </ContextMenuTrigger>
  <ContextMenuContent>
    <ContextMenuItem>
      <Text>Edit</Text>
    </ContextMenuItem>
    <ContextMenuItem>
      <Text>Delete</Text>
    </ContextMenuItem>
  </ContextMenuContent>
</ContextMenu>
```

## Props

### ContextMenu

Extends `@rn-primitives/context-menu` Root props.

| Prop | Type | Description |
|------|------|-------------|
| open | `boolean` | The controlled open state of the context menu. |
| onOpenChange | `(open: boolean) => void` | Event handler called when the open state changes. |

### ContextMenuTrigger

Extends `@rn-primitives/context-menu` Trigger props.

| Prop | Type | Description |
|------|------|-------------|
| asChild | `boolean` | Change the default rendered element for the one passed as a child. |
| disabled | `boolean` | When true, prevents context menu from opening. |

### ContextMenuContent

Extends `@rn-primitives/context-menu` Content props.

| Prop | Type | Description |
|------|------|-------------|
| className | `string` | Additional classes for styling. |
| portalHost | `string` | The host where the context menu content will be rendered. |

### ContextMenuItem

Extends `@rn-primitives/context-menu` Item props.

| Prop | Type | Description |
|------|------|-------------|
| disabled | `boolean` | When true, prevents the user from interacting with the item. |
| textValue | `string` | Optional text used for typeahead purposes. |
| className | `string` | Additional classes for styling. |
