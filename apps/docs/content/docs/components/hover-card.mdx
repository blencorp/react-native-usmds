## Usage

```tsx
import { HoverCard, HoverCardContent, HoverCardTrigger } from '@/components/ui/hover-card';
import { Text } from '@/components/ui/text';

<HoverCard>
  <HoverCardTrigger>
    <Text>Hover</Text>
  </HoverCardTrigger>
  <HoverCardContent>
    <Text>Content</Text>
  </HoverCardContent>
</HoverCard>
```

## Props

### HoverCard

Extends `@rn-primitives/hover-card` Root props.

| Prop | Type | Description |
|------|------|-------------|
| open | `boolean` | The controlled open state of the hover card. |
| onOpenChange | `(open: boolean) => void` | Event handler called when the open state changes. |
| defaultOpen | `boolean` | The open state when initially rendered. |
| openDelay | `number` | Time in milliseconds before opening. Default is `700`. |
| closeDelay | `number` | Time in milliseconds before closing. Default is `300`. |

### HoverCardTrigger

Extends `@rn-primitives/hover-card` Trigger props.

| Prop | Type | Description |
|------|------|-------------|
| asChild | `boolean` | Change the default rendered element for the one passed as a child. |

### HoverCardContent

Extends `@rn-primitives/hover-card` Content props.

| Prop | Type | Description |
|------|------|-------------|
| side | `'top' \| 'right' \| 'bottom' \| 'left'` | The preferred side of the trigger to render against. |
| sideOffset | `number` | The distance in pixels from the trigger. Default is `4`. |
| align | `'start' \| 'center' \| 'end'` | The preferred alignment against the trigger. Default is `'center'`. |
| alignOffset | `number` | Offset in pixels from the alignment axis. |
| portalHost | `string` | The host where the hover card content will be rendered. |
| className | `string` | Additional classes for styling. |

### Simple Hover Card

```tsx
<HoverCard>
  <HoverCardTrigger asChild>
    <Text className="underline cursor-pointer">Hover over me</Text>
  </HoverCardTrigger>
  <HoverCardContent>
    <Text>This is additional information that appears on hover.</Text>
  </HoverCardContent>
</HoverCard>
```
