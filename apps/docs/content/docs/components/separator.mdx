
## Props

Extends `@rn-primitives/separator` Root props.

| Prop | Type | Description |
|------|------|-------------|
| orientation | `'horizontal' \| 'vertical'` | The orientation of the separator. Default is `'horizontal'`. |
| decorative | `boolean` | When true, indicates separator is purely visual. Default is `true`. |
| className | `string` | Additional classes for styling. |
