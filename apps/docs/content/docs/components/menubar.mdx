## Usage

```tsx
import {
  Menubar,
  MenubarContent,
  MenubarItem,
  MenubarMenu,
  MenubarSeparator,
  MenubarTrigger,
} from '@/components/ui/menubar';
import { Text } from '@/components/ui/text';

<Menubar>
  <MenubarMenu>
    <MenubarTrigger>
      <Text>File</Text>
    </MenubarTrigger>
    <MenubarContent>
      <MenubarItem>
        <Text>New File</Text>
      </MenubarItem>
      <MenubarItem>
        <Text>Open</Text>
      </MenubarItem>
      <MenubarSeparator />
      <MenubarItem>
        <Text>Save</Text>
      </MenubarItem>
    </MenubarContent>
  </MenubarMenu>
</Menubar>
```

## Props

### Menubar

Extends `@rn-primitives/menubar` Root props.

| Prop | Type | Description |
|------|------|-------------|
| value | `string` | The controlled value of the menu that is currently open. |
| onValueChange | `(value: string) => void` | Event handler called when the value changes. |
| className | `string` | Additional classes for styling. |

### MenubarMenu

Extends `@rn-primitives/menubar` Menu props.

| Prop | Type | Description |
|------|------|-------------|
| value | `string` | A unique value for this menu. |

### MenubarTrigger

Extends `@rn-primitives/menubar` Trigger props.

| Prop | Type | Description |
|------|------|-------------|
| disabled | `boolean` | When true, prevents the user from opening the menu. |
| className | `string` | Additional classes for styling. |

### MenubarContent

Extends `@rn-primitives/menubar` Content props.

| Prop | Type | Description |
|------|------|-------------|
| side | `'top' \| 'right' \| 'bottom' \| 'left'` | The preferred side to render against. |
| align | `'start' \| 'center' \| 'end'` | The preferred alignment against the trigger. |
| portalHost | `string` | The host where the menu content will be rendered. |
| className | `string` | Additional classes for styling. |

### MenubarItem

Extends `@rn-primitives/menubar` Item props.

| Prop | Type | Description |
|------|------|-------------|
| disabled | `boolean` | When true, prevents the user from interacting with the item. |
| textValue | `string` | Optional text used for typeahead purposes. |
| className | `string` | Additional classes for styling. |
