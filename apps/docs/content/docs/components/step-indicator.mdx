## Styling

The StepIndicator component uses Tailwind CSS classes for styling through class-variance-authority (cva). Default styling includes:

- Container dimensions:
  - Default: 329px × 68px
  - Small: 329px × 50px
- Segment heights:
  - Default: 48px
  - Small: 32px
- Circle indicator:
  - Default: 40px × 40px
  - Small: 24px × 24px
- Primary colors for active state
- Darker primary for completed states
- Light gray for upcoming states
- Consistent spacing and gaps

## States

The step segments have three visual states:
- Complete: Darker primary color (segments before current)
- Active: Primary color (current segment)
- Default: Light gray (upcoming segments)

## Accessibility

The StepIndicator component:
