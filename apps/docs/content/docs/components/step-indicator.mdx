## Usage

```tsx
import { StepIndicator } from '@/components/ui/stepindicator';

<StepIndicator
  steps={6}
  currentStep={3}
  title="Current Step Title"
  size="default"
/>
```

## Props

### StepIndicator

Extends `View` props.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| steps | `number` | - | **Required.** Total number of steps in the sequence. |
| currentStep | `number` | - | **Required.** The current active step number (1-indexed). |
| title | `string` | - | **Required.** Title text for the current step. |
| size | `'default' \| 'small'` | `'default'` | The size variant of the step indicator. |
| labels | `string[]` | - | Optional array of labels for each step. |
| className | `string` | - | Additional classes for styling. |
