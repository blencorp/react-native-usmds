---
title: BiometricSignIn
description: A component for enabling biometric authentication options like Face ID and Touch ID.
---

## Installation

```package-install
npx usmds add biometricsignin
```

## Usage

```tsx
import { BiometricSignIn } from '@components/BiometricSignIn';
import { useState } from 'react';

function BiometricSettings() {
  const [enabled, setEnabled] = useState(false);

  return <BiometricSignIn checked={enabled} onCheckedChange={setEnabled} biometricType='faceId' />;
}
```

## Styling

The component uses Tailwind CSS for styling with these default characteristics:

```tsx
<BiometricSignIn checked={enabled} onCheckedChange={setEnabled} className='my-custom-classes' />
```

Default styling includes:

- Container: 318.88px × 50px
- Checkbox: 24px × 24px with 2px border
- Primary color for checked state
- Disabled state styling with muted colors
- Consistent padding and spacing (15px horizontal, 12px vertical)
- Rounded borders
- Base ink color for text and icons

## Accessibility

The BiometricSignIn component includes:

- `accessibilityRole="checkbox"` for semantic structure
- Proper `accessibilityState` for checked and disabled states
- Keyboard navigation support
- Clear, descriptive text labels
- WCAG-compliant color contrast
- Touch target size meets accessibility guidelines

## Test IDs

For testing purposes, the component includes:

- `testID="biometric-pressable"` on the outer pressable container
- `testID="biometric-checkbox"` on the checkbox element

## State Management

The component uses controlled state pattern:

```tsx
import { useState } from 'react';

function BiometricSettings() {
  const [enabled, setEnabled] = useState(false);

  return <BiometricSignIn checked={enabled} onCheckedChange={setEnabled} biometricType='faceId' />;
}
```
