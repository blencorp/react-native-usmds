## Usage

```tsx
import { BiometricSignIn } from '@/components/ui/biometric-signin';
import { useState } from 'react';

const [enabled, setEnabled] = useState(false);

<BiometricSignIn
  checked={enabled}
  onCheckedChange={setEnabled}
  biometricType="faceId"
/>
```

## Props

### BiometricSignIn

Extends `@rn-primitives/checkbox` Root props.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| biometricType | `'faceId' \| 'touchId' \| 'biometric'` | `'faceId'` | The type of biometric authentication. |
| checked | `boolean` | - | The controlled checked state. |
| onCheckedChange | `(checked: boolean) => void` | - | Callback when checked state changes. |
| disabled | `boolean` | `false` | Whether the component is disabled. |
| className | `string` | - | Additional classes for styling. |
