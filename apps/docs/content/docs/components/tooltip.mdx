## Usage

```tsx
import { Tooltip, TooltipContent, TooltipTrigger } from '@/components/ui/tooltip';
import { Text } from '@/components/ui/text';

<Tooltip>
  <TooltipTrigger>
    <Text>Hover</Text>
  </TooltipTrigger>
  <TooltipContent>
    <Text>Tooltip content</Text>
  </TooltipContent>
</Tooltip>
```

## Props

### Tooltip

Extends `@rn-primitives/tooltip` Root props.

| Prop | Type | Description |
|------|------|-------------|
| open | `boolean` | The controlled open state of the tooltip. |
| onOpenChange | `(open: boolean) => void` | Event handler called when the open state changes. |
| defaultOpen | `boolean` | The open state when initially rendered. |
| delayDuration | `number` | Time in milliseconds before tooltip opens on hover. Default is `700`. |
| skipDelayDuration | `number` | Duration user can hover off and back on before delay resets. |
| disableHoverableContent | `boolean` | Prevents tooltip from remaining open when hovering content. |

### TooltipTrigger

Extends `@rn-primitives/tooltip` Trigger props.

| Prop | Type | Description |
|------|------|-------------|
| asChild | `boolean` | Change the default rendered element for the one passed as a child. |

### TooltipContent

Extends `@rn-primitives/tooltip` Content props.

| Prop | Type | Description |
|------|------|-------------|
| side | `'top' \| 'right' \| 'bottom' \| 'left'` | The preferred side of the trigger to render against. Default is `'top'`. |
| sideOffset | `number` | The distance in pixels from the trigger. Default is `4`. |
| align | `'start' \| 'center' \| 'end'` | The preferred alignment against the trigger. |
| alignOffset | `number` | Offset in pixels from the alignment axis. |
| portalHost | `string` | The host where the tooltip content will be rendered. |
| className | `string` | Additional classes for styling. |
