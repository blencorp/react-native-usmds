## Props

### Tag

Extends `View` props.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| label | `string` | - | **Required.** The text content of the tag. |
| size | `'default' \| 'big'` | `'default'` | The size variant of the tag. |
| className | `string` | - | Additional classes for styling. |
