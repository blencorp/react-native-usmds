## Usage

```tsx

import { useState } from 'react';

const [text, setText] = useState('');

<Textarea
  placeholder="Enter your text here..."
  value={text}
  onChangeText={setText}
/>
```

## Props

### Textarea

Extends React Native's `TextInput` props.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| className | `string` | - | Additional classes for styling. |
| placeholderClassName | `string` | - | Additional classes for placeholder text. |
| multiline | `boolean` | `true` | Enables multiline input. |
| numberOfLines | `number` | `8` (native), `2` (web) | Number of lines to display. |
| editable | `boolean` | `true` | When false, textarea is disabled. |
| value | `string` | - | The textarea value. |
| onChangeText | `(text: string) => void` | - | Callback when text changes. |

All standard React Native TextInput props are supported.
