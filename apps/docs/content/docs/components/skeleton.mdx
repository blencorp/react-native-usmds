## Usage

```tsx
import { Skeleton } from '@/components/ui/skeleton';

<Skeleton className="h-12 w-12 rounded-full" />
```

## Props

Extends React Native `View` props.

| Prop | Type | Description |
|------|------|-------------|
| className | `string` | Additional classes for styling. |
