## Usage

```tsx
import { Select, SelectContent, SelectGroup, SelectItem, SelectLabel, SelectTrigger, SelectValue } from '@/components/ui/select';
import { Label } from '@/components/ui/label';
import { useState } from 'react';

const [value, setValue] = useState<string>();

<Select value={value} onValueChange={setValue}>
  <SelectTrigger>
    <SelectValue placeholder="Select an option" />
  </SelectTrigger>
  <SelectContent>
    <SelectGroup>
      <SelectLabel>Options</SelectLabel>
      <SelectItem label="Option 1" value="option-1">
        Option 1
      </SelectItem>
      <SelectItem label="Option 2" value="option-2">
        Option 2
      </SelectItem>
    </SelectGroup>
  </SelectContent>
</Select>
```

## Props

### Select

Extends `@rn-primitives/select` Root props.

| Prop | Type | Description |
|------|------|-------------|
| value | `string \| undefined` | The controlled value of the select. |
| onValueChange | `(value: string \| undefined) => void` | Event handler called when the value changes. |
| open | `boolean` | The controlled open state of the select. |
| onOpenChange | `(open: boolean) => void` | Event handler called when the open state changes. |
| defaultValue | `string` | The value of the select when initially rendered. |
| disabled | `boolean` | When true, prevents the user from interacting with select. |

### SelectTrigger

Extends `@rn-primitives/select` Trigger props.

| Prop | Type | Description |
|------|------|-------------|
| size | `'default' \| 'sm'` | The size of the trigger. Default is `'default'`. |
| disabled | `boolean` | When true, prevents the user from opening the select. |
| className | `string` | Additional classes for styling. |

### SelectValue

Extends `@rn-primitives/select` Value props.

| Prop | Type | Description |
|------|------|-------------|
| placeholder | `string` | The content that will be rendered when no value is selected. |
| className | `string` | Additional classes for styling. |

### SelectContent

Extends `@rn-primitives/select` Content props.

| Prop | Type | Description |
|------|------|-------------|
| position | `'popper' \| 'item-aligned'` | The positioning mode to use. Default is `'popper'`. |
| side | `'top' \| 'bottom' \| 'left' \| 'right'` | The preferred side of the trigger to render against. |
| portalHost | `string` | The host where the select content will be rendered. |
| className | `string` | Additional classes for styling. |

### SelectItem

Extends `@rn-primitives/select` Item props.

| Prop | Type | Description |
|------|------|-------------|
| value | `string` | **Required.** The value of the item. |
| label | `string` | **Required.** The label of the item. |
| disabled | `boolean` | When true, prevents the user from selecting the item. |
| className | `string` | Additional classes for styling. |
