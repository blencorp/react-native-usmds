## Usage

```tsx
import {
  Dialog,
  DialogTrigger,
  DialogContent,
  DialogHeader,
  DialogTitle,
  DialogDescription,
} from '@/components/ui/dialog';

<Dialog>
  <DialogTrigger asChild>
    <Button variant="outline">
      <Text>Open</Text>
    </Button>
  </DialogTrigger>
  <DialogContent>
    <DialogHeader>
      <DialogTitle>Title</DialogTitle>
      <DialogDescription>Description</DialogDescription>
    </DialogHeader>
  </DialogContent>
</Dialog>
```

## Props

### Dialog

Extends `@rn-primitives/dialog` Root props.

| Prop | Type | Description |
|------|------|-------------|
| open | `boolean` | The controlled open state. |
| onOpenChange | `(open: boolean) => void` | Callback when open state changes. |
| defaultOpen | `boolean` | The default open state. |

### DialogContent

Extends `@rn-primitives/dialog` Content props.

| Prop | Type | Description |
|------|------|-------------|
| className | `string` | Additional classes for styling. |
| portalHost | `string` | Optional portal host name. |

### DialogHeader

Extends `View` props.

| Prop | Type | Description |
|------|------|-------------|
| className | `string` | Additional classes for styling. |

### DialogFooter

Extends `View` props.

| Prop | Type | Description |
|------|------|-------------|
| className | `string` | Additional classes for styling. |

### DialogTitle

Extends `@rn-primitives/dialog` Title props.

| Prop | Type | Description |
|------|------|-------------|
| className | `string` | Additional classes for styling. |

### DialogDescription

Extends `@rn-primitives/dialog` Description props.

| Prop | Type | Description |
|------|------|-------------|
| className | `string` | Additional classes for styling. |
