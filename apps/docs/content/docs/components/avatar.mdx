---
title: Avatar
description: An image element with a fallback for representing the user.
---

## Installation

```package-install
npx usmds add avatar
```

## Usage

```tsx
import { Avatar, AvatarImage, AvatarFallback } from '@components/Avatar';
import { Text } from 'react-native';

<Avatar alt="User Avatar">
  <AvatarImage source={{ uri: 'https://github.com/user.png' }} />
  <AvatarFallback>
    <Text>JD</Text>
  </AvatarFallback>
</Avatar>
```

## Examples

### Default Avatar

```tsx
<Avatar alt="John Doe">
  <AvatarImage source={{ uri: 'https://github.com/johndoe.png' }} />
  <AvatarFallback>
    <Text>JD</Text>
  </AvatarFallback>
</Avatar>
```

### Custom Size

```tsx
<Avatar alt="Custom Size" className="h-16 w-16">
  <AvatarImage source={{ uri: 'https://github.com/user.png' }} />
  <AvatarFallback>
    <Text className="text-lg">CS</Text>
  </AvatarFallback>
</Avatar>
```

### Custom Background

```tsx
<Avatar alt="Custom Background">
  <AvatarImage source={{ uri: 'https://github.com/user.png' }} />
  <AvatarFallback className="bg-primary">
    <Text className="text-primary-foreground">CB</Text>
  </AvatarFallback>
</Avatar>
```

### Multiple Avatars

```tsx
<View className="flex-row gap-4">
  <Avatar alt="User 1">
    <AvatarImage source={{ uri: 'https://github.com/user1.png' }} />
    <AvatarFallback>
      <Text>U1</Text>
    </AvatarFallback>
  </Avatar>
  <Avatar alt="User 2">
    <AvatarImage source={{ uri: 'https://github.com/user2.png' }} />
    <AvatarFallback>
      <Text>U2</Text>
    </AvatarFallback>
  </Avatar>
</View>
```

## Styling

The Avatar component comes with default styling:

- Root: `relative flex h-10 w-10 shrink-0 overflow-hidden rounded-full`
- Image: `aspect-square h-full w-full`
- Fallback: `flex h-full w-full items-center justify-center rounded-full bg-muted`

You can customize the appearance using:
- `className` prop for additional Tailwind classes
- Custom background colors for the fallback
- Text styling for the fallback content
- Custom sizes through width and height classes

## Accessibility

The Avatar component:
- Requires an `alt` prop for screen reader accessibility
- Provides fallback content when images fail to load
- Maintains WCAG color contrast requirements
- Uses semantic HTML elements
